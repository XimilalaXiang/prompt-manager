# Knowledge Base

The knowledge base allows you to store reference articles and notes alongside your prompts.

## Features

- **Create articles** with title, content, and category
- **Favorite** important articles for quick access
- **Archive** articles you don't need regularly but want to keep
- **Search** through your knowledge base
- **Organize** with categories

## Use Cases

- Store prompt engineering best practices
- Keep API documentation snippets
- Save useful AI conversation patterns
- Document your prompt templates and strategies
