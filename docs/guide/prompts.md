# Prompt Management

## Creating Prompts

Each prompt consists of:

- **Title**: A descriptive name
- **Content**: The prompt text (supports system prompt and user prompt)
- **Category**: Optional category for organization
- **Tags**: Optional tags for filtering
- **Favorite**: Star important prompts for quick access

## Version History

Every time you edit a prompt, the previous version is automatically saved. You can:

- View the full version history of any prompt
- Compare changes between versions
- Roll back to any previous version

## Search and Filter

The prompt list supports:

- **Text search**: Search by title and content
- **Category filter**: Filter by category
- **Favorites filter**: Show only starred prompts
- **Pagination**: Navigate through large collections

## Categories

Organize prompts into categories:

- Create categories with name and optional color
- Assign prompts to categories
- Filter the prompt list by category
- Categories can be renamed or deleted (prompts are preserved)
