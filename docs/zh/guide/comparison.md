# 多模型对比

Prompt Manager 的核心功能之一是并排对比多个 AI 模型的响应。

## 使用方法

1. 选择 2 个或更多 AI 配置
2. 输入提示词
3. 同时发送给所有模型
4. 并排查看带流式传输的响应

## 流式模式

对比页面支持 SSE（Server-Sent Events）流式传输。每个模型的响应会实时显示，无需等待所有模型完成。

## AI 自动评分

收到响应后，可以使用自动评分功能让 AI 模型评价和打分每个响应。评分考虑：

- 与提示词的相关性
- 响应的质量和准确性
- 完整性和详细程度

## 支持的提供商

支持任何 OpenAI 兼容的 API 端点，包括：

- OpenAI（GPT-4、GPT-4o、GPT-3.5 等）
- Anthropic（通过兼容代理）
- Google Gemini（通过兼容代理）
- DeepSeek
- Groq
- 本地模型（Ollama、LM Studio 等）
- 任何其他 OpenAI 兼容 API 的提供商

## 保存对比结果

对比结果可以保存为对话会话以供后续查看。每个会话存储：

- 使用的提示词
- 所有模型的响应
- 评分和备注
