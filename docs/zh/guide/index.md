# 什么是 Prompt Manager？

Prompt Manager 是一个自托管的 AI 提示词管理 Web 应用。适合 AI 深度用户：

- 组织和版本控制提示词
- 并排对比多个 AI 模型的响应
- 安全存储 API 配置（加密）
- 在提示词旁维护知识库
- 自托管，零外部依赖

## 架构

- **后端**：Go（Gin 框架）+ SQLite 数据库
- **前端**：React 18 + TypeScript + Tailwind CSS（新野兽派设计风格）
- **部署**：单个 Docker 镜像（约 20MB），同时服务 API 和前端

所有数据存储在单个 SQLite 文件中，启用 WAL 模式支持并发访问。

## 核心功能

| 功能 | 说明 |
|---|---|
| 提示词管理 | 完整生命周期管理，支持搜索和收藏 |
| 版本历史 | 自动版本控制，支持回滚 |
| 多模型对比 | 同时发送给多个模型 |
| SSE 流式 | 实时流式响应 |
| AI 自动评分 | 自动化模型响应评分 |
| 知识库 | 参考文章和笔记 |
| 对话管理 | 保存和回顾 AI 对话 |
| 导入/导出 | JSON 格式备份恢复 |
| 安全 | AES-256-GCM + JWT + bcrypt |
