# 提示词管理

## 创建提示词

每个提示词包含：

- **标题**：描述性名称
- **内容**：提示词文本（支持系统提示和用户提示）
- **分类**：可选的分类标签
- **标签**：可选的过滤标签
- **收藏**：标记重要提示词方便快速访问

## 版本历史

每次编辑提示词时，上一个版本会自动保存。你可以：

- 查看任意提示词的完整版本历史
- 对比不同版本间的变化
- 回滚到任意历史版本

## 搜索和筛选

提示词列表支持：

- **文本搜索**：按标题和内容搜索
- **分类筛选**：按分类过滤
- **收藏筛选**：仅显示收藏的提示词
- **分页**：浏览大量提示词集合

## 分类管理

将提示词组织到分类中：

- 创建带名称和可选颜色的分类
- 将提示词分配到分类
- 按分类筛选提示词列表
- 可重命名或删除分类（提示词会保留）
